# demo-ml-classification
A demo to help me understand the concept of classification in ML. Specifically, Binary Logistic Regression. 

Project Overview: This Streamlit application explains binary classification, showing how computers make "yes or no" decisions by sorting data into two specific buckets.

Applied Scenario: A practical demonstration predicting whether a community water pump will fail. It combines multiple data points (pump age, daily water usage, and months since the last inspection) to calculate the exact probability of a breakdown and determine if a mechanic must be dispatched.
